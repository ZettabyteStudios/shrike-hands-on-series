# Vicharak Shrike Hands-On Series

This repository contains the code examples and projects used in the **Vicharak Shrike Hands-On Series**, a practical learning series focused on embedded systems and digital hardware design.

The tutorials explore how microcontrollers and FPGAs work together in modern embedded systems using the **Vicharak Shrike development platform**.

The series covers topics such as:

- Programming the RP2040 using MicroPython
- Designing digital circuits using Verilog
- Simulating hardware designs
- Interfacing microcontrollers with FPGA logic
- Building complete embedded systems

Each episode includes code examples, experiments, and projects that can be run directly on the Shrike board.
